# 更新日志

本项目的所有重要变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.8.1] - 2026-04-11

### 变更

- **存档页交互降噪**：移除顶部状态横幅与配对提示条，空闲状态只保留卡片高亮、连线和必要的右下角错误反馈。
- **加载过渡重做**：存档页初次进入改为同构骨架屏配合分层淡入，云存档区、同步栏、槽位卡片与备份区的出现节奏更平滑。

### 修复

- 修复进入存档页时会短暂闪出“未找到原版存档 / 未找到模组存档”等空状态的问题，现在只有在真实数据加载完成后才显示空态。
- 修复存档页加载态与真实内容尺寸不匹配导致的明显跳变，骨架布局现在贴合真实卡片、操作按钮与备份区域。

## [0.8.0] - 2026-04-11

### 新增

- **预设整合包（Bundle）**：将预设配置与所有关联 Mod 打包为一个 `.zip` 整合包（内含 `.spm` 清单文件），一键分享给好友或社区。
- **导入整合包**：在预设页点击「导入整合包」或直接拖拽整合包到窗口，即可预览包含的 Mod 列表并一键导入；若检测到同名 Mod 冲突，会弹出冲突解决面板，逐个 Mod 选择「跳过」或「替换」。
- **从模组库一键保存预设**：模组库页面新增「保存为预设」按钮，将当前已启用的 Mod 集合快速保存为新预设，支持自定义名称与描述。
- **拖拽导入整合包**：将整合包 `.zip` 文件拖入应用窗口，自动识别是否为整合包并跳转预设页完成导入流程。

### 变更

- **移除旧版 JSON 导出**：预设页不再提供仅导出 JSON 配置的功能，统一使用「分享整合包」，确保接收方无需额外下载 Mod。
- **删除预设确认弹窗**：删除预设时改为使用应用内 `ConfirmDialog` 组件，替代系统原生 `window.confirm`，视觉风格统一。
- **冲突解决交互优化**：冲突 Mod 的「跳过 / 替换」选择从下拉框改为分段式按钮（Pill Toggle），操作更直观。
- **侧边栏预设列表实时刷新**：导入或删除预设后，侧边栏的预设选择器会立即同步更新。

### 修复

- 修复导出整合包时将空 Mod 目录（无实际文件）打入 ZIP 的问题，空目录现在会被自动跳过。
- 修复模组库批量检查更新时界面卡顿的问题，改为异步逐条返回结果。
- 修复 i18n 模板中 `{count}` 和 `{path}` 占位符未正确渲染的问题。

## [0.7.1] - 2026-04-07

### 修复

- **更新代理支持**：应用内的版本检查与更新下载现在会复用设置页里配置的 HTTP / HTTPS / SOCKS5 代理，让更新请求与其他联网功能保持一致的代理路径。
- **更新时保留桌面快捷方式偏好**：Windows MSI 升级现在会继承旧安装中的桌面快捷方式选择。若旧版本安装时没有启用桌面快捷方式，升级完成后会移除本次升级新生成的桌面快捷方式及其注册表标记。

## [0.7.0] - 2026-04-07

### 新增

- **Nexus 插图预览**：发现页现在可以直接预览模组插图，减少来回打开详情页的成本。
- **多人影响标记**：模组库与预设页的模组选择列表现在会为 `affects_gameplay = true` 的模组显示“影响多人”标签。

### 变更

- **默认深色主题**：首次启动或没有本地主题偏好时，应用默认使用深色主题。
- **界面打磨**：优化模组库筛选栏、折叠侧边栏与版本徽标的视觉层级和交互反馈。
- **存档槽位对齐**：存档管理页左右两侧槽位改为按槽位号固定 1:1 对齐，跨环境连接关系更直观。

### 修复

- 修复 UTF-8 BOM 与 UTF-16 编码的 mod JSON 无法正确识别，导致元信息读取失败的问题。
- 修复预设页在模组状态变化后未及时刷新，以及左侧预设列表滚动时脱离标题区的问题。
- 修复存档管理页在拖拽窗口尺寸后连接线穿过卡片、错位或误导槽位对应关系的问题。

## [0.6.1] - 2026-04-05

### 新增

- **云存档差异审视工作台**：新增专门的本地/Steam Cloud 真实差异审视窗口，支持 delta 风格 diff、两侧文件元数据查看、文本文件内联编辑，以及本地与云缓存之间的精确复制。
- **Steam Cloud 同步文档**：新增 `docs/steam-cloud-sync-notes.md`、`docs/remotecache-vdf-reference.md` 和 `docs/save-file-reference.md`，系统说明 Steam Cloud 缓存机制、`remotecache.vdf` 和长期进度相关存档字段。

### 变更

- 云缓存准备流程现在会在写入云侧文件后对齐文件 mtime，并重建 `remotecache.vdf`，让 Steam 下次启动时看到一份更自洽的本地缓存快照。
- 启动游戏前的云存档差异提示现在可以直接跳转到存档页，并自动打开云差异审视窗口。
- 云差异审视窗口默认隐藏 `*.backup` 项，先把真正影响长期进度的核心文件呈现出来。

### 修复

- 修复误删 `history/*.run.backup` 这类 Steam 仍会在启动同步阶段读取的文件后，导致 Slay the Spire 2 启动崩溃的问题；危险的 backup cleanup 路径现已禁用。
- 修复审视窗口与确认弹窗被应用顶层拖拽区、标题区遮挡，导致界面看起来“卡住”的问题。
- 修复在 Steam 或游戏仍在运行时直接改写云缓存导致状态错乱的问题；云侧写操作现在会要求先关闭 Steam 和游戏。

## [0.6.0] - 2026-04-03

### 新增

- **Steam 云存档同步**：自动发现当前 Steam 账号的云存档目录（`userdata/<Account_ID>/2868840/remote`），支持一键「上传至云端」与「下载至本地」全量同步
- **云存档安全机制**：每次云端操作前自动创建完整备份，杜绝误覆盖风险
- **打开云存档目录**：云存档面板新增文件夹按钮，可直接在资源管理器中打开 Steam 远程存档目录
- **毛玻璃设计重构**：全应用级别的磨砂玻璃材质（`backdrop-filter: blur(40px) saturate(180%)`），覆盖侧边栏、面板、卡片和对话框
- **浅色模式优化**：全新暖色调浅色主题，采用 Warm Amber 强调色和暖白色玻璃表面，告别生硬白底
- **预设系统视觉升级**：重构了预设管理栏与细节面板，新增了带动画效果的高级下拉菜单（`custom-select`）和统一工具栏

### 变更

- 存档页布局重构为双列网格 + 顶部云存档横幅，连接线从卡片右/左边缘水平延伸
- 所有云存档相关文案完成 i18n 国际化（中/英双语）
- 后端错误信息改为 i18n key 输出，前端自动翻译展示
- 存档覆写逻辑改为「清空内容后复制」，避免 Steam 文件锁导致的权限错误

## [0.5.0] - 2026-03-27

### 新增

- **模组备注 / 别名** (#2)：支持为已安装的模组添加自定义备注或别名，方便辨别。备注以彩色标签显示在模组卡片上，支持在搜索栏中按备注搜索。数据保存在本地 localStorage 中。
- **回车执行搜索**：模组库与发现页的搜索现在仅在按下回车键后触发，避免输入过程中频繁刷新和资源占用。输入时输入框右侧会显示“Enter ↵”提示。

## [0.4.2] - 2026-03-24

### 新增

- **版本展示与更新体验**：侧边栏底部显示当前版本号胶囊；有可用更新时亮起绿色指示点；点击可跳转设置页
- **设置页「关于」板块**：展示当前版本号、更新状态徽标，以及「检查更新」/「立即更新」按钮
- **统一更新上下文** (`UpdateContext.tsx`)：集中管理更新状态（检查中、可用、下载中、重启中、出错），供侧边栏、设置页和弹窗共享
- **可配置的自动备份保留数量**：新增设置项，控制每个存档槽位保留的自动备份数（1–50，默认 5）；手动备份始终保留
- **profile 目录级同步**：存档操作（备份、同步、恢复）现在会处理完整的 profile 目录，包括 `replay/` 子文件夹
- **毛玻璃吸顶标题栏**：设置页、存档页和预设页的标题栏在滚动时固定在顶部，带毛玻璃虚化效果
- **打开文件夹按钮**：每个存档槽位卡片增加文件夹图标，可快速打开对应目录

### 变更

- 存档历史分组现在**默认折叠**，界面更简洁
- 重构 `UpdateChecker.tsx`，改为使用共享的 `UpdateContext`
- 改进存档迁移确认弹窗：增加可视化槽位选择器和更清晰的备份说明
- 备份历史改为按时间线分组展示

### 修复

- 存档同步现在会复制 profile 目录下的所有文件，避免因缺少 `replay/` 文件夹导致存档异常
- 自动备份裁剪遵循配置的保留数量，在每次同步和存档守护周期后自动执行
- 版本号移至「启动游戏」按钮下方，视觉更整洁
- 修复预设页标题栏与窗口控制按钮重叠的问题
