## 📋 Mod Installation Tips

### Using SlaySP2Manager (Recommended)

1. Open **SlaySP2Manager** and go to the **Discover** tab
2. Search for the mod you want
3. Click **Download** — the app handles everything automatically
4. Enable/disable mods from the **Library** tab

### Manual Installation

If you prefer to install mods manually:

1. Download the mod archive (`.zip`, `.7z`, etc.)
2. Locate your game directory:  
   Steam → Right-click *Slay the Spire 2* → Manage → Browse Local Files
3. Create a `mods` folder in the game root directory (if it doesn't exist)
4. Extract the mod's `.dll` and `.pck` files directly into the `mods` folder
5. Launch the game — it will auto-detect and load the mods

> ⚠️ **Warning:** First-time mod installation may cause save data issues. **Always back up your saves first!**  
> Save path: `C:\Users\<YourName>\AppData\Roaming\SlayTheSpire2\steam\<SteamID>`
