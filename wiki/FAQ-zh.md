## 💬 常见问题

### Q：SlaySP2Manager 是什么？

**A：** SlaySP2Manager 是一款免费、开源的桌面端《杀戮尖塔 2》Mod 管理器。它自动化了 Mod 安装、冲突检测、存档管理等流程——全部集成在一个简洁的界面中。基于 Rust + Tauri + React 构建。

---

### Q：SlaySP2Manager 安全吗？

**A：** 安全！应用完全开源（[在 GitHub 上查看代码](https://github.com/wakaka6/SlaySP2Manger)）。它不会直接修改游戏文件——仅管理 `mods` 文件夹和存档文件。应用还会在任何高风险操作前自动创建备份。

---

### Q：支持 macOS 或 Linux 吗？

**A：** 目前，SlaySP2Manager 仅支持 **Windows 10/11 (x64)**。macOS 和 Linux 支持可能在未来添加。

---

### Q：SlaySP2Manager 与 Mega Crit 或 Nexus Mods 有关联吗？

**A：** 没有。SlaySP2Manager 是一个独立的社区项目，未获得 Mega Crit（游戏开发商）或 Nexus Mods 的官方背书。

---

## 📥 安装与设置

### Q：从哪里下载 SlaySP2Manager？

**A：** 从 [Releases 页面](https://github.com/wakaka6/SlaySP2Manger/releases) 下载最新的 `.msi` 安装包。

---

### Q：应用无法自动检测游戏目录怎么办？

**A：** 进入 **设置** 手动设置游戏路径。查找方法：
1. 打开 Steam → 库
2. 右键 **杀戮尖塔 2** → 管理 → 浏览本地文件
3. 复制路径并粘贴到 SlaySP2Manager 的游戏路径设置中

常见游戏路径：
- `C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2`
- `D:\SteamLibrary\steamapps\common\Slay the Spire 2`

---

### Q：Windows SmartScreen 阻止了安装程序，安全吗？

**A：** 安全。因为应用未使用昂贵的代码签名证书，Windows SmartScreen 可能会显示警告。点击 **"更多信息"** → **"仍要运行"** 即可继续。应用是开源的，你可以自行查看代码。

---

### Q：首次启动出现报错怎么办？

**A：** 请确认：
1. 设置中的游戏路径设置正确
2. 游戏至少启动过一次（以确保存档目录存在）
3. 游戏根目录下有 `mods` 文件夹（应用会在需要时自动创建）
4. 运行设置页面中的内置 **诊断检查** 功能

## 📦 Mod 管理

### Q：如何使用 SlaySP2Manager 安装 Mod？

**A：** 有两种方式：

**方式 1 — 从 Nexus Mods 下载（推荐）：**
1. 进入 **发现** 页面
2. 搜索你想要的 Mod
3. 点击 **下载** —— 搞定！

**方式 2 — 从本地文件导入：**
1. 进入 **模组库** 页面
2. 点击 **导入** 按钮（或拖拽文件）
3. 选择包含 Mod 的 `.zip` 或 `.7z` 文件
4. 确认安装预览

---

### Q：可以一次导入多个 Mod 吗？

**A：** 可以！你可以在文件选择器中选择多个文件，或选择一个包含多个 Mod 压缩包的文件夹。SlaySP2Manager 支持批量导入，每个 Mod 独立处理——某个 Mod 导入失败不会影响其他 Mod。

---

### Q：如何启用或禁用 Mod？

**A：** 在 **模组库** 页面中，每个 Mod 都有一个开关。点击即可启用或禁用。被禁用的 Mod 会从活动的 `mods` 文件夹中移出，游戏不会加载它们。

---

### Q：导入 Mod 后侧边栏的 Mod 数量没有更新。

**A：** 这个问题在最新版本中已修复。如果数量显示不准确，请尝试：
1. 切换到其他页面再切回
2. 重启应用
3. 从 [Releases 页面](https://github.com/wakaka6/SlaySP2Manger/releases) 更新到最新版本

---

### Q：支持哪些压缩格式？

**A：** SlaySP2Manager 支持：
- ✅ **ZIP** — 完全支持
- ✅ **7z** — 完全支持
- ⚠️ **RAR** — 有限支持，可能需要将 Mod 重新打包为 ZIP/7z
- ✅ **文件夹** — 也可以直接导入已解压的 Mod 文件夹

---

### Q：看到"检测到冲突"警告是什么意思？

**A：** 当两个或更多 Mod 包含同名文件时会发生冲突。SlaySP2Manager 会在做出任何更改之前提醒你。你可以选择：
- 移除冲突的某个 Mod
- 继续安装（新 Mod 的文件将覆盖旧的）

---

### Q：可以撤销 Mod 安装吗？

**A：** SlaySP2Manager 会在高风险操作前自动创建备份。你可以在 **存档管理** 区域浏览和恢复备份。此外，你也可以直接在模组库中禁用或卸载 Mod。

---

## 🔑 Nexus Mods 集成

### Q：如何获取 Nexus Mods API Key？

**A：** SlaySP2Manager 内置了引导教程：
1. 进入 **设置**
2. 按照 **API Key 设置引导** 操作——它会指引你创建免费 Nexus Mods 账号并生成 API Key
3. 将 API Key 粘贴到设置字段中

API Key 是 **免费** 的，只需一个 Nexus Mods 账号。

---

### Q：粘贴了 API Key 但发现页面显示错误。

**A：** 请检查：
1. 确保 API Key 完整复制（没有多余的空格）
2. 检查网络连接
3. 如果你在防火墙后面或网络受限地区，尝试在设置中配置 **代理**
4. Nexus Mods API 可能有临时故障——稍后再试

---

### Q：不用 API Key 能下载 Mod 吗？

**A：** **发现** 页面需要 API Key 才能搜索和下载 Nexus Mods 上的 Mod。但你始终可以：
- 在浏览器中手动从 [nexusmods.com](https://www.nexusmods.com) 下载 Mod
- 使用本地导入功能将它们导入 SlaySP2Manager

---

### Q：下载速度很慢怎么办？

**A：** 尝试：
1. 如果网络受限，在 **设置** 中配置代理
2. 检查网络连接速度
3. 在非高峰时段下载
4. Nexus Mods 免费用户有下载速度限制——升级高级账号可以解除限制

---

## 💾 存档管理

### Q：安装 Mod 会损坏存档吗？

**A：** **首次** 在 STS2 中安装任何 Mod 时，存在已知的存档丢失风险。SlaySP2Manager 通过以下方式降低风险：
- 在任何操作前自动备份存档
- 清晰地分离原版和 Mod 存档槽位
- 提供便捷的备份恢复功能

**请务必保持存档备份！**

---

### Q：存档文件在哪里？

**A：** 默认存档路径：
```
C:\Users\<你的用户名>\AppData\Roaming\SlayTheSpire2\steam\<你的SteamID>
```

SlaySP2Manager 可以自动检测此路径。如果无法检测，请在设置中手动配置。

---

### Q：存档配对/同步是怎么工作的？

**A：** 存档配对可以将一个原版存档槽位与一个 Mod 存档槽位关联起来。启用自动同步后：
- 根据修改时间双向同步变更
- 可视化连线显示配对关系
- 点击连线上的 × 按钮可解除关联

这对于同时维持原版和 Mod 两套存档流程非常有用。

---

### Q：不小心覆盖了存档，能恢复吗？

**A：** 可以！SlaySP2Manager 会在每次高风险操作前自动创建备份。进入存档管理区域，浏览 **备份列表** 即可恢复之前的版本。

---

## 🌐 代理与网络问题

### Q：如何配置代理？

**A：** 进入 **设置** → **代理设置**：
1. 打开代理开关
2. 选择代理类型（HTTP、HTTPS 或 SOCKS5）
3. 输入代理地址和端口
4. 如有需要，输入用户名/密码进行认证
5. 点击 **测试连接** 验证
6. 保存设置

---

### Q：我在网络受限的地区，还能使用 SlaySP2Manager 吗？

**A：** 可以！在设置中配置代理，通过代理服务器路由网络请求。也可以：
1. 手动从 Nexus Mods 或其他来源下载 Mod
2. 将文件传输到你的电脑
3. 使用本地导入功能安装

---

### Q：应用完全无法连接 Nexus Mods。

**A：** 排查步骤：
1. 检查网络连接
2. 尝试在浏览器中访问 [nexusmods.com](https://www.nexusmods.com)
3. 如有需要，在设置中配置代理
4. 检查防火墙或杀毒软件是否阻止了应用
5. 尝试设置页面中的内置 **诊断检查** 功能

---

## ⚡ 性能与兼容性

### Q：Mod 操作时应用变慢或无响应。

**A：** 应用会阻止并发操作以避免文件冲突。如果你一次导入很多 Mod，批量处理可能需要一些时间。请等待当前操作完成后再开始新操作。

---

### Q：使用 Mod 会影响 Steam 成就吗？

**A：** 是的，**Mod 可能会禁用 Steam 成就**。这是游戏层面的限制，不是 SlaySP2Manager 导致的。未来可能会出现"成就解锁器"Mod（初代游戏中有类似的 Mod）。

---

### Q：合作多人模式下能用 Mod 吗？

**A：** 可以，但有限制：
- 所有玩家必须使用 **完全相同的游戏版本**
- 所有玩家必须安装 **相同的 Mod**（相同版本号）
- 标记为 `affects_gameplay: true` 的 Mod 必须所有人都安装
- 纯外观类 Mod 可能不需要所有人都安装

---

## 🎮 游戏端 Mod 问题

### Q：安装 Mod 后游戏崩溃了。

**A：** 尝试以下步骤：
1. 在 SlaySP2Manager 中禁用最近安装的 Mod
2. 检查该 Mod 是否需要 **BaseLib** 或其他前置依赖
3. 确保 Mod 与你的游戏版本兼容
4. 检查是否与其他已安装的 Mod 冲突
5. 尝试禁用所有 Mod，然后逐个启用来排查

---

### Q：Mod 文件应该放在哪里？

**A：** Mod 文件（`.dll` 和 `.pck`）应直接放置在游戏根目录下的 `mods` 文件夹中：
```
<游戏目录>/mods/
  ├── mod_name.dll
  ├── mod_name.pck
  └── mod_manifest.json  （可选）
```

**不要** 为每个 Mod 创建子文件夹，除非 Mod 明确要求这样做。SlaySP2Manager 会自动处理所有这些。

---

### Q：安装 Mod 导致进度重置，如何解锁所有内容？

**A：** 如果安装 Mod 导致进度重置：
1. 在游戏主菜单按 **`~`** 键（波浪线键，`1` 左边）打开控制台
2. 输入 `unlockall` 并按回车
3. 即可解锁所有角色、卡牌、遗物和游戏模式

要启用完整的控制台功能，在 `settings.save` 文件中添加 `"full_console": true`。

---

### Q：如何完全移除所有 Mod？

**A：** 两种方式：
1. **使用 SlaySP2Manager：** 在模组库页面禁用或卸载所有 Mod
2. **手动操作：** 删除游戏目录中 `mods` 文件夹内的所有文件

---

### Q：游戏提示我以"Mod 模式"重启，这正常吗？

**A：** 正常！当 STS2 检测到 `mods` 文件夹中有 Mod 时，可能会提示你以 Mod 模式重启。这是正常行为，说明 Mod 正在被正确加载。
